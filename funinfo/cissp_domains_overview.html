Part 1 – 8 CISSP Domains as “Core Questions”

Domain Core Question It Answers
1. Security & Risk Management“What are our goals, obligations, and risk appetite?”
2. Asset Security“What do we have, how sensitive is it, and how do we handle it from creation to destruction?”
3. Security Architecture & Engineering“How do we design and build secure systems and controls?”
4. Communication & Network Security“How do we securely connect systems and protect data in motion?”
5. Identity & Access Management (IAM)“Who gets access, and how do we control and verify it?”
6. Security Assessment & Testing“How do we check that our security measures are working?”
7. Security Operations“How do we run security every day and respond to incidents?”
8. Software Development Security“How do we build and maintain software securely?”

Part 2 – Plain English Summaries (1 sentence each)

  • Security & Risk Management – Sets the rules: policies, risk appetite, compliance, ethics, and leadership.
  • Asset Security – Identifies valuable data and defines how to classify, protect, store, and destroy it.
  • Security Architecture & Engineering – Designs secure systems, using principles like defense-in-depth and encryption.
  • Communication & Network Security – Protects information as it moves across networks and systems.
  • Identity & Access Management – Controls who can log in and what they can do.
  • Security Assessment & Testing – Tests, audits, and scans to make sure defenses work as intended.
  • Security Operations – The day-to-day security work: monitoring, responding to incidents, backups, patching.
  • Software Development Security – Builds security into software from the start (secure coding, SDLC, DevSecOps).
